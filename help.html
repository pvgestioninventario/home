Bienvenido al Centro de Ayuda de Punto de Venta

En este documento se describen las funcionalidades principales del sistema y cómo utilizarlas de manera eficiente.

El sistema permite realizar ventas tanto de contado como a crédito, lo que lo hace ideal para negocios como tiendas de ropa, electrodomésticos, tecnología, farmacias y abarrotes.
